     <section>
       <p>
         MedlinePlus offers services for developers. All are available at no cost and without requiring any registration or licensing. These services allow developers to build on and point to information from MedlinePlus. MedlinePlus has hundreds of health topic pages that bring together information from NIH, other U.S. government agencies, and reputable health information providers. <a href="https://www.nlm.nih.gov/medlineplus/healthtopics.html">Health topic pages</a> cover a wide range of conditions and wellness issues, and include overviews, information on symptoms and treatments, recent health news, and clinical trials.
         
       </p>    </section>
   <section>
     <h2>MedlinePlus Web Service</h2> 
     	<p>
     	  MedlinePlus offers a <a href="https://www.nlm.nih.gov/medlineplus/webservices.html">search-based Web service</a> that provides access to MedlinePlus health topic data in XML format. The service accepts keyword searches as requests and returns relevant health topics in ranked order and is available for <a href="https://www.nlm.nih.gov/medlineplus/healthtopics.html">English</a> and <a href="https://www.nlm.nih.gov/medlineplus/spanish/healthtopics.html">Spanish</a> health topics. Refer to the MedlinePlus <a href="https://www.nlm.nih.gov/medlineplus/webservices.html">Web service</a> page to learn more.  
     	</p>
     </section>
     <section>
       <h2>MedlinePlus Connect</h2> 
     	<p>
     	  <a href="https://www.nlm.nih.gov/medlineplus/connect/overview.html">MedlinePlus Connect</a> allows health organizations and health IT providers to link patient portals and electronic health record (EHR) systems to content from MedlinePlus.gov. MedlinePlus Connect accepts code-based requests for information on diagnoses (problem codes), medications, and lab tests, and returns related MedlinePlus information. Connect is available as a <a href="https://www.nlm.nih.gov/medlineplus/connect/application.html">Web application</a> or a <a href="https://www.nlm.nih.gov/medlineplus/connect/service.html">Web service</a>.  It can deliver responses in English or Spanish. Read more about <a href="https://www.nlm.nih.gov/medlineplus/connect/overview.html">MedlinePlus Connect</a>. 
     	  
     	</p>
     </section>
     <section>
       <h2>MedlinePlus XML Files</h2> 
     	<p>
     	  MedlinePlus publishes health topic XML files. These XML files contain records for all <a href="https://www.nlm.nih.gov/medlineplus/healthtopics.html">English</a> and <a href="https://www.nlm.nih.gov/medlineplus/spanish/healthtopics.html">Spanish</a> health topics. Each health topic record includes data elements associated with that topic.  These XML files allow you to download and use virtually all text and links that appear on MedlinePlus health topic pages. For details, see the MedlinePlus <a href="https://www.nlm.nih.gov/medlineplus/xml.html">XML files</a> page.
     	</p>
     </section>
   <section><p>If you have questions about any of these MedlinePlus products for developers, please <a href="http://apps.nlm.nih.gov/medlineplus/contact/index.cfm?lang=en&from=https://www.nlm.nih.gov/medlineplus/fordevelopers.html">contact us</a>.</p></section>
